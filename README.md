# 💜 Portfólio - Marina Felix

Bem-vinda(o) ao meu portfólio profissional 🚀  
Este projeto foi desenvolvido com o objetivo de apresentar minhas habilidades como desenvolvedora FullStack, destacando meus projetos, conhecimentos e evolução na área de tecnologia.

---

## ✨ Sobre o Projeto

Este portfólio foi criado com foco em:

- Design minimalista
- Experiência do usuário
- Identidade visual moderna
- Animações e efeitos visuais tecnológicos
- Código limpo e organizado

A proposta foi desenvolver uma interface elegante, com fundo em gradiente e efeitos de partículas que trazem uma estética tech e contemporânea.

---

## 🛠 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  
- tsParticles (efeito de partículas)

---

## 🎯 Funcionalidades

✔️ Seção de apresentação  
✔️ Projetos desenvolvidos  
✔️ Informações de contato  
✔️ Fundo animado com partículas interativas  
✔️ Layout responsivo
